HomePortals Site XML Reference

This document describes the tags used in a HomePortals Site document. The site.xml document is used by the HomePortals Accounts system and provides information about an Account's pages. This document must be located on each Account/<usernname>/ folder.

 

<site>

Description:
This is the root node of the document. All tags must be children of this tag.

Attributes:
None

<title>

Description:
Title for the site described in the current XML. Currently this title does not used. This tag should appear only once per document.

Attributes:
None

<pages>

Description:
This tag is used to enclose the region of the document where the site pages are listed. May appear only once per document. This tag should only contain <page> tags as children.

Attributes:
None

<page>

Description:
Identifies a HomePortals page on the site.

Attributes:

Name Type Description

href

String

Required. Filename of the HomePortals page. The page should be an existing file in the layouts subdirectory of the current account directory.

title

String

Required. Page Title. This is the label that will be shown on the SiteMap module..
private Boolean Optional. Determines if a page is private. Private pages may only be accessed by the account owner and are only displayed on the SiteMap. Default is false.

<defaultPage>

Description:
This tag encloses the template to use when adding new pages to this site. When the user adds a page using the SiteMap, the contents of this node will be used as the source of the new page.

Attributes:
None

<catalogs>

Description:
This tag is used to enclose the region of the document where the site catalogs are listed. May appear only once per document. This tag should only contain <catalog> tags as children.

Attributes:
None

<catalog>

Description:
Identifies a catalog to which the account is subscribed. Catalogs are xml documents that contain references to modules and published pages. The catalogs are used by the Control Panel to display available modules for the user to add to a page.

Attributes:

Name Type Description

href

String

Required. URL of the catalog file.

name

String

Required. Catalog title.
