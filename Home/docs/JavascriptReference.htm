HomePortals JavaScript Reference

This document describes the Javascript functions provided by HomePortals. These functions are contained in the Main.js file.

 

Name / Parameters

Description

h_Hide(secName)

Sets the css display property to “none” for the element with ID secName.

h_Show(secName,display)

Sets the css display property to display for the element with ID secName. If display is not provided, then defaults to “block”.

h_CollapseSection(secID)

Collapses a container element. On a collapsed container only the Title bar is visible. secID is the ID of the module which container should be collapsed.

h_ExpandSection(secID)

Expands a container element. secID is the ID of the module which container should be expanded.

h_IsVisible(secName)

Returns True if the element with ID secName has its css display property different than “none”.

h_IsSectionExpanded(secID)

Returns True if the container of the module with ID secID is expanded.

h_ToggleSection(secID)

Toggles the state of the container of the module with ID secID between Expanded and Collapsed.

h_ToggleSectionVisibility(secID)

Toggles the visibility state of the element with ID secID

h_SetSectionTitle(secID,title)

Changes the title of the container of the module with ID secID to title

h_GetSectionTitle(secID)

Returns the title of the container of the module secID

h_Print(secID,srcID)

If srcID is missing, then prints the contents of the container of the module secID. If srcID is provided, then prints the contents of srcID using the title of the module secID.

h_callServer(server,method,sec,params,rcv)

Makes an asynchronous call a remote server using the parameters given.
server identifies the remote process or file that will be called. This can be a relative or absolute path. External addresses will be called via http requests made by a gateway on the server.
method will be passed as a variable to the target server.
sec is the ID of the element in which the returning output of the call will be displayed. If empty or not found, the the received output will be discarded.
params is a collection representing attributes or parameters that will be passed to the server side element.
rcv is the name of a function that will be called when the call to the server-side function has returned.

h_resizeToClient(sec,offset)

Resizes a section to fit the browser’s window height. The height of the element sec will be changed to the current browser’s window height minus offset.
This function uses the clientHeight property to obtain the height of the window.

h_setLoadingImg(secID)

This function changes the contents of secID with an animated gif of a loading bar. You may use this function when waiting for the server to return with a user interface element.

h_raiseEvent(objectName, eventName, args)

This function is defined only when event listeners have been declared on the configuration XML using <event> tags.
This function can used to raise a framework event by any module in the application.
